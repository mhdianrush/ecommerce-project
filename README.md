## 🛠️ **Prerequisites**
- A running instance of the API (default: `http://localhost:8000`).
- Tools to test the API:
  - [Postman](https://www.postman.com/)
  - `curl` (command line tool)
  - HTTP client library (e.g., Axios, Fetch).
